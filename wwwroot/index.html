Each lesson opens with a TypeScript → C# analogy, explains one concept, and ships with a live editor wired to a real Roslyn-backed C# compiler running entirely in your browser.

  • Lessons 1–10 cover the language (types, classes, LINQ, generics).
  • Lessons 11–18 cover the .NET platform (async, DI, minimal APIs, EF Core, tests).
  • Lessons 19–30 cover real-world conventions you'll see in production code.
  • Lesson 31 is a capstone project you'll build in a separate terminal.
